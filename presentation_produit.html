Architecture

Comment une séance devient une calorie

Six étapes, six responsabilités séparées. Aucune couche ne refait le calcul d'une autre.

01

Moteur de calcul C

calculs.c contient les formules pures — IMC, calories, MET — compilées en DLL. Aucune validation ici : juste des nombres qui entrent, et des nombres qui sortent.

02

Pont Python ctypes + SQLite

module.py valide chaque entrée avant d'appeler la DLL, lit la table d'activités depuis SQLite, et persiste les séances.

03

API REST Flask

Une seule porte d'entrée. Toutes les réponses suivent le même format JSON — aucun client ne parle directement à la base de données.

04

Dashboard web Chart.js

Trois graphiques en lecture seule, rafraîchis automatiquement toutes les 30 secondes. Zéro calcul côté navigateur.

05

Application desktop C# · .NET 8

Cinq écrans WinForms qui communiquent uniquement via HttpClient. Le C# affiche — il ne calcule jamais.

06

Tests & Benchmark Pytest

10 tests automatisés couvrent les calculs et la persistance. Le benchmark ci-dessus compare trois façons d'appeler la même DLL.

Installation

En marche en moins de 5 commandes

Sur une machine Windows avec Python 64-bit et GCC installés.

$git clone https://github.com/freezerkg/TFE_wyatt
$gcc -m64 -shared -o calculs.dll calculs.c -lm -Wl,--export-all-symbols
$pip install flask pytest
$python app.py
Running on http://127.0.0.1:5000

Le flag -m64 doit correspondre à l'architecture de ton Python (32 ou 64 bits) — la cause la plus fréquente de blocage est documentée en premier dans le guide.

À propos

Pourquoi ce projet

Dans mon apprentissage de l'informatique dans le présent, rien ne résume mieux ce que j'ai appris, l'IA. je n'ai appris quasiment aucun compétence utile cette année durant mes cours d'informatique, ce projet est la représentation de ça.

Nous avons passés une année à recevoir des cours pour apprendre quasiment chaque langage de programmation présent dans le projet et même après tout cela ce projet n'est pas le miens. Je ne ressent aucune passion pour ce projet et même si j'en avait, le limite de temps ridicule qui m'avait été imposé ne m'aurait même pas permis de donner libre cours à ma passion et mes compétence.

Voici ce qui défini mon présent dans l'informatique, ne pas apprendre suffisamment pour pouvoir coder tout seul et ne pas avoir reçu la possibilité de faire le travail sans utiliser une IA.
